# IoT Smart Campus using Secure Federated Learning

This repository contains my portfolio copy of a team project developed as part of an academic/internship project.

## Project Overview

This project focuses on privacy-preserving anomaly detection in a smart campus IoT environment using Secure Federated Learning. Data collected from IoT devices is used to train machine learning models while preserving data privacy.

## My Contributions

- Collected and preprocessed data from IoT devices deployed in the smart campus.
- Performed data cleaning and feature preparation for machine learning.
- Developed and trained the machine learning model for anomaly detection.
- Evaluated model performance and improved prediction accuracy.
- Collaborated with team members during project development and testing.

## Technologies Used

- Python
- Machine Learning
- Federated Learning
- TensorFlow
- IoT
